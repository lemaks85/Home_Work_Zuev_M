```plantuml

[*] --> ОжиданиеОповещения : Пользователь зарегистрирован

ОжиданиеОповещения --> ПолучениеОповещения : Получено оповещение о пожаре

ПолучениеОповещения --> [*] : Пользователь проигнорировал оповещение

@enduml
```
