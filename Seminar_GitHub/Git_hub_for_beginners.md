> `git --version` - дял проверки версии после установки.

> Зоздаем попку `code` и в ней создаем папку `project` далее открываем эту папку через `vs code`.

> `git init` - иннициализация.
 
> `git add .` - добавляет все файлы.  
> `git add имя файла с расширением` - добовляет только указанный файл.
Изначально файл поподает в `Changes`. Но после команды `add` он переходит в `Staged Changes`. 

> `git commit -m "комментарий".

> `git config --global user.email "тут почта"`.  
`git config --global user.name "тут имя"`.
`git config --list` - проверка данных git.  
Для продолжения списка нажимать Enter, когда появится `END` нажать `Q`.   
Также данные можно проверить в файле `.gitconfig`.

> `git clone ссылка на репазиторий` 

> `git log`
